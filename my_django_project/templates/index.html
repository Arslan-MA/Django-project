Моя идея Web-прилежение аптека

Целевая аудитория: кто будет использовать ваше приложение, какие у них нужды и проблемы

Любой человек кто нуждается в лекарстве. Также поиск дешевого лекарства

Функционал: какие задачи должно решать приложение, какие функции должно предоставлять пользователям.

Использовать моё приложение будут для покупки и поиска лекарств, поиска аптек по карте, сравнение цен, оплата онлайн, напоминание о принятии лекарств и т.д.

Распределение контента и функционала по веб-страницам: на какой странице какой контент и функции будут располагаться.

  1. Главная страница: Банеры с рекламой продукта, популярные товары, баланс на счету, поиск лекарств по названию и т.д.
  2. История: история покупок и т.д.
  3. Будильник: напоминание о принятии лекарств
  4. Профиль: личная информация, кошелёк и т.д.

Дизайн: как приложение будет выглядеть с точки зрения дизайна. Минимум 3 ссылки на сторонние сайты, которые по дизайну будут схожи с вашим проектом.
